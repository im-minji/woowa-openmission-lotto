로또 웹 프로젝트

우아한 테크코스 오픈미션

이 프로젝트는 3주차의 로또 미션을 Spring Boot 웹 API로 확장하는 오픈미션 프로젝트입니다.

사용자가 로또 번호를 기록하고 관리할 수 있는 웹 페이지를 구현하는 것을 목표로 합니다.

MyLotto(나만의 로또): 저장해둔 나만의 로또를 관리하는 페이지

RandomLotto: 로또를 랜덤 생성할 수 있는 페이지

PurchasedLotto: 구매한 로또를 관리하는 페이지
